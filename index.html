Tic-Tac-Toe

Tic Tac Toe

A game of Tic Tac Toe playable by two players.

Using HTML/CSS and Javascript/JQuery.

Features:

- Game keeps score of wins/ties, with a reset option.

- Alternates between 'X' starting and 'O' starting.

Shopping List

Shopping List

A dynamic list, intended for shopping for groceries.

Timer with start/stop and reset functions to track how quickly you gather the items on your list.

Features:

- Using Javascript/JQuery, users can enter items to be kept in a dynamic list.

- Items can be deleted and checked off as obtained.

- A timer written with JS/JQuery counts up each second.

Topsify

Topsify

Using the Spotify API, search a band name to bring up a list of albums with pictures.

Choose an album to bring up a list of song titles. Updatable at any time.

Click the play button on any song to bring up a spotify integrated player to play the song (Spotify account required).

Features:

- Interacts with Spotify API to pull artist, album and track information.

- Uses an integrated player that is hidden until needed to play each song.

Hot or Cold

Hot or Cold

A number guessing game.

Guess a number and receive feedback on whether the number is higher or lower.

Features:

- Number of guesses is tracked.

- Each previous guess is logged and shown, with color coordination to indicate if the old guess was higher or lower.

French Revolution Quiz

French Revolution Quiz

A five question quiz on the French Revolution.

Answer multiple choice questions and get your score at the end.

Features:

- Each question has an accompanying explanation of correct and incorrect answers.

- Tracks number of questions correct, and presents a final score at the end, with a bonus for getting all 5 correct.

- Quiz can be restarted at any time.

Mini Street Fighter

Mini Street Fighter

A small animation demo using Ryu from Street Fighter.

Ryu and a clone face off to trade cool poses and Haduokens.

Features:

- Each Ryu can pose and launch Haduokens.

- HTML/CSS and Javascript/JQuery is used to achieve each animation.
